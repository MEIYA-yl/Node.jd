## Express：
官方网址 https://www.expressjs.com ：基于 Node.js 平台，快速、开发、极简的 web 开发框架。
1. 特色：
	- web 应用：提供一系列特性，便于开发者创建web与移动设备应用。
	- API：提供HTTP快捷方法以及组合connect中间件。
	- 性能：在 node 的基础上，扩展了web应用所需的基本功能。

2. 安装：
	- $ npm install express --save
